2. Inline Level element

inline요소는 한줄에 표시되며, width, height, margin-top/bottom을 쓸 수 없다.

a 태그를 가장 자주 쓰지 span안에 넣어주면 inline화 된다 모두 inline level element라서 한줄에 표시된다

3. inline-block

  1. inline이지만 block에서 사용되는 속성을 활용할 수 있다
  2. width, height, margin-top, margin-bottom값을 지정할 수 있다
  3. margin-left, margin-right는 inline도 가능하다
